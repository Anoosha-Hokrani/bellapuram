# Project bella sagaram — Anoosha Hokrani

A GitHub Pages site showcasing 10 smart-city solutions powered by algorithmic systems: waste, parking, energy, ports, transport, water, emergency, housing, markets, and logistics.

## Deploy on GitHub Pages
1. Create a new repo (e.g., `swarna-nagar`).
2. Upload `index.html` and `styles.css` to the root of the repo.
3. Commit and push.
4. In the repo: Settings → Pages → Source = `main` branch, Root (`/`) → Save.
5. Your site will be live at `https://<your-username>.github.io/swarna-nagar/`.

## Customize
- Change text and colors in `index.html` and `styles.css`.
- Replace the avatar image with a real photo URL or local file.
